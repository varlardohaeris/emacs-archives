Provides an Emacs CAT client for controlling Yaesu FT991A.

Under construction... eventually it will support a catmacs Major Mode

Basic stuff works, including ...

* Frequency
* Mode
* RF Gain
* AF Gain
* RF Attenuator
* QMB Recall
* Band Select
* Band Up/Down
* Encorder Up/Down
* Channel Up/Down
* Noise Blanker
* LED/TFT Dimmer
